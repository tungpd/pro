KLineChart Pro Sample Application

Demonstrating financial charting with drawing tools and technical indicators

Features Demonstrated:

  • Drawing Tools: Use the drawing bar to add technical analysis overlays
  • Technical Indicators: Click the indicator button to add MA, RSI, MACD, etc.
  • Time Periods: Switch between different timeframes
  • Themes: Toggle between light and dark themes
  • Symbol Switching: Change between different stocks
  • Settings: Customize chart appearance and behavior

Controls:

  • Zoom: Mouse wheel or pinch gestures
  • Pan: Click and drag on the chart
  • Crosshair: Hover over price bars for details
  • Screenshot: Export chart images
